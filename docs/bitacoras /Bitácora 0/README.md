# Bitácora 0 – Inicio del Proyecto

## Comprensión del problema
El sistema que buscamos desarrollar consiste en automatizar una cortina de uso industrial, la cual actualmente requiere de una operación manual. La idea principal es que la cortina pueda subir y bajar de manera controlada, segura y automática, considerando que puede tener diferentes medidas y pesos dependiendo del lugar donde se instale.

El proyecto implica pensar cómo hacer que la cortina tenga un funcionamiento adecuado, es decir, que se mueva en el momento correcto, se detenga en los límites establecidos y pueda ser operada por una persona de forma sencilla. Aunque en la situación problema se describen de manera general los componentes del sistema, todavía no comprendemos a profundidad cómo funcionan ni cómo se implementan técnicamente; sin embargo, se entiende que deberán combinarse partes mecánicas, eléctricas y de control para lograr que el sistema funcione de manera correcta y confiable. De esta manera, podemos entender el proyecto como un sistema mecatrónico en el que varios elementos interactúan entre sí, desde la detección de condiciones, el control de decisiones y la actuación mecánica, hasta la interacción que hay con el usuario.

## Autoevaluación inicial
- Sensores: Nuestro conocimiento sobre sensores es básico. Sabemos que sirven para detectar ciertas condiciones o posiciones, por ejemplo, si la cortina ya llegó a su punto máximo o mínimo, pero aún no conocemos los tipos específicos ni cómo se configuran o se seleccionan.
- Actuadores: Entendemos que los actuadores son elementos que permiten el movimiento del sistema, como el mecanismo que hace que la cortina suba o baje. Sin embargo, todavía no tenemos experiencia utilizándolos.
- Controlador (LOGO): Nuestro conocimiento del LOGO es muy básico, solo hemos trabajado con él en la práctica que hicimos en clase, donde aprendimos cosas básicas, pero aún no dominamos ni su uso ni sus posibilidades completas.
- Integración: Sabemos que la integración se refiere a que todos los componentes del sistema trabajen de manera conjunta; sin embargo, aún no sabemos cómo se realiza este proceso de forma práctica.

## Retos percibidos
¿Cuál crees que será el mayor reto del proyecto?
Nuestro mayor reto será lograr que todos los elementos del sistema funcionen de manera coordinada, especialmente considerando el poco conocimiento inicial que tenemos acerca de los elementos con los cuales vamos a trabajar. También será un desafío comprender cómo integrar correctamente todas las partes para que el sistema sea seguro, confiable y cumpla con los requisitos solicitados.

## Expectativas de aprendizaje
¿Qué esperas aprender al finalizar el curso?
Esperamos aprender cómo se analiza y se desarrolla un sistema mecatrónico desde el inicio, así como identificar y seleccionar sensores y actuadores adecuados. Por otra parte, esperamos adquirir mayor confianza en el uso del controlador LOGO y comprender completamente qué significa la integración de un sistema, no solo de manera teórica, sino aplicada a un problema de la vida real, como lo es el de la cortina industrial.
