# Bitácora – Semana 4 Max

## Tema de la semana
(Control / HMI)

## Actividades realizadas
En esta semana se probo el funcionamiento del sistema ya instalado, verificamos el funcionamiento de los interlocks y que cumpliera con la condición prohibida puesta en el sistema. Teniendo nuestro proyecto montado dimos una introducción pequeño a los HMI y como estos pueden complemenetar a nuestro proyecto, por falta de tiempo no lo pudimos implementar el HMI al proyecto.

## Decisiones de ingeniería
| Decisión | Alternativas | Justificación |
|--------|-------------|---------------|
|Realizar un modelo en CAD e imprimirlo para el soporte del motor. |Usar cinchos o cinta para amarrar el motor.|Realizamos un modelo en CAD para implementar el diseño al proyecto y además se viera mas profesional y estético.|

## Problema técnico encontrado
Nuestra cortina de repente dejo de funcionar el motor, identificamos que los relé estaban mal conectados por movimientos que se hacian.

## Solución aplicada
Ordemos un poco el cableado y checamos continuidad con un multímetro para ver si una conexión se habia salido.

## Conexión con el curso
¿Qué concepto de MR2022 aplicaste esta semana?

Durante esta semana aplicamos conceptos de automatización y control mediante el uso de un PLC (LOGO!) para gestionar el funcionamiento del sistema. Se verificó la implementación de interlocks de seguridad que permiten bloquear el movimiento del sistema cuando ocurre una condición prohibida. Además, se revisó de forma introductoria el uso de interfaces HMI, comprendiendo cómo estas pueden utilizarse para monitorear y controlar procesos automatizados en sistemas industriales.
